# Encuentra Repositorios full stack

El desarrollador Full Stack es un perfil muy cotizado en el entorno empresarial, y muy útil si estás planteando arrancar tu startup o desarrollar un producto propio. Te permite tener una visión de todas las áreas que forman un proyecto web, tanto Back End cómo Front End.
